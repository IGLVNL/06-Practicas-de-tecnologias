Variables

Una variable es un espacio en memoria, dedicado a contener algún tipo de dato

Declaracion de una variable

Existen 3 formas de declarar una variable

Var
Let
Const

El Scope, es el ambito de la variable, y este determina el alcance de la misma, asi como su accesibilidad.

Declarar una variable significa darle un nombre, y definir su tipo.

Inicializar una variable significa darle un valor a la misma.

Uno puede declarar una variable sin definir el ambito de la misma (Let o var), pero implicitamente, Javascript las reconoce como Var, por lo que siempre se recomienda declarar el ambito mas usado, en general Let.

El Hoisting, es un comportamiento que hace que las variables y funciones declaradas, se "eleven" al principio del bloque donde se las declara, lo que hace que cuando se las declare, sin importar en donde, estas se comportan como si se las declarasen al principio del bloque, aunque su asignacion si se realice en la ubicacion correspondiente dentro del código.

Operadores logicos

Son aquellos operadores que devuelven true o false, si se cumple una condición.

AND (&&)
OR (||)
NOT (!)

Cabe aclarar, que los operadores lógicos, solo funcionan comparando valores o resultados Booleanos.

Cuando decimos "Devolver", hacemos referencia a que luego de realizar ciertas operaciones preestablecidas, se tiene un resultado especifico.

Estas operaciones, se pueden combinar de manera casi infinita. Por ejemplo: Console.log((A < B) && (C >= D) || (!E <= F) && (G > H))
